# Tareas para los estudiantes - Para 30 enero

## Entregable Principal

1. **Consultar y descargar 3 bases de datos en formato CSV**
   - Descargar de la plataforma de datos abiertos del gobierno
   - Pueden ser de Kaggle, Google Dataset Search o SIMEM
   - Requisito: que tengan datos numéricos, ordinales y descripciones

2. **Elegir una sola base de datos** (una de las tres)
   - Realizar un EDA (Exploratory Data Analysis)
   - Incluir análisis Cuantitativo, Cualitativo y Gráfico

3. **Dar explicaciones de negocio** a partir de los datos elegidos
   - Ser generosos con las descripciones proporcionadas

4. **Formato de entrega**
   - El EDA debe hacerse a través de un notebook de Python
   - Enviar a través de repositorio de GitHub con todos los integrantes del equipo
   - Incluir un enlace donde estén las 3 bases de datos consultadas

5. **Criterios de evaluación**
   - Se calificará el notebook de Python con las explicaciones
   - Es importante incluir alguna metodología de ciencia de datos

6. **Ponderación**
   - Esto corresponde a la nota del primer módulo: **25%**

### Introducción a Streamlit:
**Streamlit** es un framework de código abierto de Python para científicos de datos e ingenieros de AI/ML que permite:
- Entregar aplicaciones de datos interactivas
- Crear aplicaciones con solo algunas líneas de código
- Una forma más rápida de construir y compartir aplicaciones de datos
